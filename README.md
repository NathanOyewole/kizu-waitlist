# KIZU

> **Build what hurts.**

Kizu is a signal intelligence tool that scans Reddit, X, and Discord to detect verified user pain points and high-value market gaps. It replaces manual market research with an automated feed of bleeding-neck problems, ensuring you only build what users are already asking for.

---

### Stack
- **Framework:** Next.js 15 (App Router)
- **Database:** Supabase (PostgreSQL)
- **Styling:** TailwindCSS v3
- **Language:** TypeScript
